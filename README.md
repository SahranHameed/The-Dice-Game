# 🎲 Dice Game — Boss Level Challenge 01

🚀 Live Demo: https://sahranhameed.github.io/The-Dice-Game/

## 📌 About the Project

A fun two-player dice rolling game built with **HTML, CSS, and JavaScript** as part of the MERN Stack Development training — Boss Level Challenge 01.

---

## 🎮 How to Play

1. Open `index.html` in your browser
2. Click **🎲 Roll Dice** button
3. Both players roll simultaneously
4. The player with the **higher number wins!**
5. Scores are tracked automatically
6. Click **🔄 Reset Scores** to start fresh

---

## ✨ Features

- 🎲 Random dice roll for both players
- 🏆 Winner highlight with glow effect
- 📊 Score tracking (Player 1 Wins / Draws / Player 2 Wins)
- 🎨 Smooth dice rolling animation
- 🔄 Reset scores button
- 📱 Responsive design

---

## 🛠️ Technologies Used

| Technology | Purpose |
